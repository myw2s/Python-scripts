<Environment>
 - This script is only developed for Windows platform

<Installation>
1. Open 'Setup' folder
2. Double click "python-2.7.6.msi" to install
3. Double click "pyserial-2.7.win32.exe" to install COM port library

<How to use>
1. Ensure the device is factory new
2. Open 'Script' folder
3. Check the COM port number in control panel
4. Open 'serialConfig.py' and make sure the COM port number is correct
5. Run 'getDeviceInfo.py' and wait until complete
6. Open 'log' folder
7. Open 'deviceInfo.txt' to view the device info
